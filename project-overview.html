Guess The Athlete Game

A browser game where the computer picks an athlete from NBA, NFL, MLB, or NHL and the player asks yes/no questions to identify them in under 25 questions.

Play Game View Repository

Core Gameplay

Ask yes/no questions, narrow the hidden player, then submit a final guess.

Multi-Sport Dataset

Covers major North American leagues with current and retired player entries.

Guided Deduce Mode

A fast elimination flow that reduces candidates based on your answers.

No Build Step

Single-file app in index.html using plain HTML/CSS/JS.

Project Snapshot

  • Hidden player from NBA/NFL/MLB/NHL each round
  • Strict yes/no response style for in-game questions
  • Question count + leaderboard to encourage efficient guessing
  • Single-page game design with no build step
